# EffekseerForCocos2d-x

This is a wrapper to play effects in Cocos2d-x

At first, please read Help.

- Players
This directory contains code files to use EffekseerForCocos2d-x

- Help
This directory contains help about EffekseerForCocos2d-x

- Samples
This directory contains samples ofEffekseerForCocos2d-x

■　EffekseerForCocos2d-x

Cocos2d-xでエフェクトを使用するためのラッパーです。

まずは、ヘルプを読みましょう。

・Players
EffekseerForCocos2d-xを使用する時に使うコードが保存されています。

・Help
EffekseerForCocos2d-xの使い方が保存されています。

・Samples
EffekseerForCocos2d-xリのサンプルが保存されています。
